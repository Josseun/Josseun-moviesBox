# 🎬 MovieBox

MovieBox is a modern movie streaming web application built with React, TypeScript, and TMDB API.

## Features

- Trending Movies
- Popular Movies
- Top Rated
- Search
- Movie Details
- Favorites
- Responsive Design

## Tech Stack

- React
- TypeScript
- TailwindCSS
- React Router
- TMDB API

## Getting Started

npm install

npm run dev
